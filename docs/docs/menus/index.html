Menus

Menus are a way to provide contextual actions within a given context. For example, on the mobile version of this site, when you click the “hamburger” menu, you are presented with a menu instead of seeing them displayed in the navigation. Menus are a way to hide links until a user actually needs to perform an action.

Example Usage:

  • Edit
  • Delete
  • Archive
  • Menu Link #1
  • Menu Link #2

Code Example:

    
        <ul class="menu">
            <li><a href="#">Edit</a></li>
            <li><a href="#">Delete</a></li>
            <li><a href="#">Archive</a></li>
        </ul>

        <ul class="menu">
            <li><a href="#">Menu Link #1</a></li>
            <li><a href="#">Menu Link #2</a></li>
        </ul>
